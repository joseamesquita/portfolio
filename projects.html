Vacation

Goal: Create a webpage about a place I would like to go on vacation using the tools below:

  1. Include a container to pad all content.
  2. Use a Jumbotron to hold a title, and an introductory sentence or slogan.
  3. Include a section with blockquotes to contain what previous visitors to this area have said.
  4. Use several helper classes to add color that emphasizes certain areas of the page.
